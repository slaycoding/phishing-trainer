Phishing Trainer

Интерактивный тренажёр, который помогает научиться отличать фишинговые сообщения от безопасных.  
Проект полностью работает в браузере и не требует установки серверной части.


Цели проекта

- Тренировать навык распознавания фишинговых писем и сообщений.
- Показать типичные признаки фишинга (подозрительные ссылки, срочные требования, ошибки и т.д.).

Как это работает

1. На экране показывается текст сообщения (письма).
2. Пользователь выбирает один из вариантов:
	- "This is phishing" - это фишинг;
	- "This is safe" - это безопасное сообщение.
3. Приложение показывает:
	- правильно ли сделан выбор;
	- объяснение, почему сообщение считается фишинговым или безопасным.
4. В конце показывается итоговая статистика:
	- количество правильных ответов;
	- процент верных решений;
	- список всех задач с результатами.


Технологии

- "HTML" - структура страницы.
- "CSS" - оформление, тёмная тема и кастомные шрифты.
- "JavaScript" - логика переключения сообщений, проверка ответов, подсчёт результатов.
- "GitHub Pages" - хостинг статического сайта (запуск из браузера по ссылке).
